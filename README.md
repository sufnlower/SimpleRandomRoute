This web application is Javascript and HTML. 

1. Takes the browser location and a distance.
2. Heuristicaly creates GPS coordinates that should result in a route of the specified distance, starting at your location and returning to your location.
3. Gives a link to a google map with the route on it.

Best results using on mobile where the link will open the google maps app and you can start directions.
